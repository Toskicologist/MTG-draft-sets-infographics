Where do the prices come from? (TCG Mid & TCGcsv)

TCG Mid is TCGplayer's Mid price — the median of the current Near Mint / Lightly Played listings for a card. It's the "typical asking price" figure, distinct from TCG Market (an average of recent completed sales, which is what the Scryfall API reports).

TCGplayer's own pricing API is closed to new apps, so the Mid figure here comes from TCGcsv — a free, community-run service that mirrors TCGplayer's public pricing data (low / mid / high / market) and republishes it as plain daily JSON. It refreshes roughly once a day (~20:00 UTC). This page uses a snapshot of that data taken shortly after each refresh; the banner above shows exactly when that snapshot was built, and every export is stamped with the same time so you always know how fresh the numbers are.

Each card is matched to its TCGplayer product by Scryfall's tcgplayer_id (an exact ID match, not a name guess). Cards with no Mid for the chosen printing — or that aren't Modern-legal — are flagged, never estimated. Switch the Price figure selector to TCG Market (live) for real-time sale-based prices straight from Scryfall instead.
