# Archive

This folder contains previous versions and experimental iterations of the MTG color wheel visualizations.

## Contents

- **lorwyn-color-wheel-v*.html** - HTML versions from v1 through v12.23 (superseded by v13)
- **lorwyn-color-wheel*.jsx** - Earlier React component versions
- **eoe-archetypes.jsx** - Experimental Explorers of Ixalan archetype visualization
- **color-wheel-preview.html** - Early preview version
- **lorwyn-color-wheel-v*.png** - PNG snapshots of earlier versions
- **netlify** - Netlify deployment configuration
